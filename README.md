# .github
UniVPN — удобный Telegram-бот, который позволяет быстро купить VPN и получить стабильное подключение без сложных настройек. Промокод START активирует 3-дневный тестовый период, чтобы оценить качество сервиса, скорость и стабильность. Подходит для повседневного использования, путешествий и работы в сети.
